# Synergen REST Responses for Tasks

The Task data type will be transformed and returned as the value of the `content` JSON property as follows:

## Task Resource

```json5
{
  "id": "",
  "title": "",
  "details": "",
  "type": "",
  "complete": false,
  "data": {},
  "_links": {
    "self": ""
  }
}
```
